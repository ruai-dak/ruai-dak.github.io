What is a sensor

A sensor is a device that detects and responds to some type of input from the physical environment. The main components of a sensor are a receptor, transducer, signal conditioning electronics, and an output indicator or alarm. The receptor portion collects the environmental input such as light, heat, or sound. The transducer then converts this input into a common form that can be easily measured and processed, such as electrical signals.

What is an actuator

An actuator is a component of a machine that produces force, torque, or displacement, when an electrical, pneumatic or hydraulic input is supplied to it in a system (called an actuating system). The effect is usually produced in a controlled way.[1] An actuator translates such an input signal into the required form of mechanical energy..
